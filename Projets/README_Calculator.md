______Calculatrice Scientifique avec Tkinter____

1-Description

Cette application est une calculatrice graphique développée en Python avec la bibliothèque Tkinter.
Elle permet d’effectuer des opérations mathématiques de base via une interface simple, moderne et responsive.
Projet réalisé dans le cadre d’un apprentissage en développement Python et interfaces graphiques.

2-Objectif : comprendre la création d’interfaces graphiques (GUI) en Python et la gestion des événements utilisateur.

3-Fonctionnalités: Addition,Soustraction,Multiplication,Division

-Saisie des nombres (0–9)
-Ajout de nombres décimaux
-Bouton AC pour effacer l’écran
-Bouton pour supprimer le dernier caractère
-Calcul du résultat
-Gestion des erreurs (division par zéro, syntaxe invalide)
-Interface
-Boutons dynamiques avec grille adaptative
-Affichage en temps réel des opérations
-Limites du projet
    .Utilisation de eval() 
    .Pas encore de fonctions scientifiques avancées (sin, cos, log, etc.)
    .Pas de gestion clavier (uniquement souris)
-Améliorations possibles
    .Ajouter des fonctions scientifiques (sin, cos, √, etc.)

    .Ajouter le support clavier
